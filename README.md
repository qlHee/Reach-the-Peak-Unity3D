# Reach the Peak - Unity3D

## 游戏介绍

这是一个第三人称视角的闯关游戏。玩家需要控制角色跟随金币的指引，躲避障碍物，获取金币，不断前进，最终到达终点。

**游戏特色：**
- 🎮 第三人称3D平台跳跃游戏
- 🏔️ 以"登峰"为主题的关卡设计
- 🪙 金币收集与路径指引系统
- ⚡ 多样化的障碍物和机关
- 🎵 沉浸式背景音效

这是我在计算机图形学与游戏开发课程中的个人作业项目。

## 项目结构

```
Reach-the-Peak-Unity3D/
├── Assets/                    # Unity资源文件
│   ├── Scenes/               # 游戏场景
│   ├── Scripts/              # C#脚本
│   ├── Animations/           # 角色动画
│   ├── Audio/                # 音频文件
│   ├── Effects/              # 粒子特效
│   └── Prefabs/              # 预制体
├── ProjectSettings/          # Unity项目设置
├── Packages/                 # Unity包管理
└── README.md                 # 项目说明
```

## 快速开始

1. **克隆项目**
   ```bash
   git clone https://github.com/qlHee/Reach-the-Peak-Unity3D.git
   ```

2. **打开项目**
   - 使用 Unity 2022.3 LTS 打开项目文件夹
   - 等待Unity导入所有资源

3. **运行游戏**
   - 打开 `Assets/Scenes/Main Scene.unity`
   - 点击播放按钮开始游戏

## 游戏控制

- **移动**: WASD / 方向键
- **视角转动**: QE
- **跳跃**: 空格键
