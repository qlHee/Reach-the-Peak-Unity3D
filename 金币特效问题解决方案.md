## 🎯 使用批量配置工具

### 第一步：打开工具窗口

在 Unity 编辑器顶部菜单栏：
```
Tools → 批量配置金币音效特效
```

会弹出一个配置窗口。

### 第二步：配置参数

在弹出的窗口中：

1. **拾取音效**：从 `Assets/Audio` 文件夹拖入音效文件
2. **音效音量**：调整滑块（默认 0.8）
3. **拾取特效预制体**：从 `Assets/Effects` 文件夹拖入特效预制体（如 `FX_CoinCollect`）
4. **特效销毁时间**：设置特效持续时间（默认 2 秒）

### 第三步：应用配置

点击窗口中的大按钮：
```
应用到场景中所有金币
```

工具会自动：
- ✓ 查找场景中所有带有 `Collectible` 组件的对象
- ✓ 为它们统一配置音效和特效
- ✓ 显示配置结果

### 第四步：保存场景

配置完成后，记得保存场景：
```
Ctrl+S (Windows) 或 Cmd+S (Mac)
```

## 🎨 让金币特效更明显的技巧

### 技巧1：调整粒子特效参数

如果特效还是不够明显，可以调整粒子系统参数：

**在 Unity 中选择特效预制体（如 `FX_CoinCollect`），修改：**

#### 增加粒子数量
```
Emission → Bursts → Count: 30-50（原来是 20）
```

#### 增加粒子大小
```
Start Size: 0.3-0.5（原来是 0.2）
```

#### 使用更鲜艳的颜色
```
Start Color: 亮金黄色（RGB: 255, 220, 0）
```

#### 添加发光效果
1. 在 `Renderer` 模块中
2. 将 `Material` 改为发光材质（如 `Default-Particle`）
3. 或创建自定义发光材质

#### 增加特效持续时间
```
Duration: 1.5-2.0（原来是 1.0）
Start Lifetime: 1.2-1.5（原来是 0.8）
```

### 技巧2：添加多层特效

创建更复杂的特效组合：

1. **内层**：小而快的金色粒子（向上喷射）
2. **外层**：大而慢的光环（向外扩散）
3. **闪光**：瞬间的白色闪光

**操作方法：**
1. 创建一个空对象，命名为 `FX_CoinCollect_Enhanced`
2. 在其下创建多个 `Particle System` 子对象
3. 分别配置不同的效果
4. 将整个父对象制作成预制体

### 技巧3：添加音效反馈

确保音效清晰可闻：
- 音效文件选择清脆的"叮"声或"铃"声
- 音量设置为 0.8-1.0
- 使用未压缩的 `.wav` 格式

## 🔍 调试特效问题

### 检查清单

如果特效还是不显示，按以下步骤检查：

#### 1. 检查特效预制体是否正确
```
选择金币对象 → Inspector → Collectible (Script)
查看 "Pickup Effect" 字段是否有预制体
```

#### 2. 检查特效是否被实例化
```
运行游戏 → 拾取金币 → 查看 Console 面板
应该看到：✓ 播放金币拾取特效：FX_CoinCollect，位置：...
```

#### 3. 检查特效位置
```
如果看到日志但看不到特效，可能是：
- 特效生成在摄像机视野外
- 特效颜色太暗或透明
- 特效太小
```

#### 4. 检查粒子系统设置
```
打开特效预制体，检查：
- Looping: 应该是 ✗（不循环）
- Play On Awake: 应该是 ✓（自动播放）
- Start Color: Alpha 应该 > 0（不透明）
- Emission → Bursts: Count 应该 > 0
```

#### 5. 临时测试方法

在场景中手动放置特效预制体：
1. 将 `FX_CoinCollect` 拖入 Hierarchy
2. 点击 Play 按钮
3. 观察特效是否播放

如果手动放置也看不到，说明特效本身有问题，需要重新配置。

## 💡 额外建议

### 1. 为不同类型的金币配置不同特效

如果你有多种金币（如普通金币、大金币、特殊金币）：
- 创建不同的特效预制体（如 `FX_CoinNormal`、`FX_CoinBig`）
- 使用批量工具分别配置

### 2. 添加金币旋转动画

让金币更吸引人：
- 已有 `CoinRotation.cs` 脚本
- 确保金币对象上有此组件

### 3. 添加金币闪光效果

使用 Unity 的 Emission（自发光）：
1. 选择金币的材质
2. 启用 Emission
3. 设置发光颜色为金黄色
