# 🎯 音效与特效配置 - 快速参考卡

## 📥 第一步：下载音效资源

### 推荐网站
- **Freesound.org** - https://freesound.org/
- **Mixkit** - https://mixkit.co/free-sound-effects/
- **Unity Asset Store** - 搜索 "Free Sound Effects"

### 需要的音效
| 音效类型 | 用途 | 格式建议 |
|---------|------|---------|
| 背景音乐 | BGMPlayer | .mp3 |
| 跳跃音效 | PlayerController | .wav |
| 落地音效 | PlayerController | .wav |
| 拾取音效 | Collectible | .wav |
| 陷阱音效 | Hazard | .wav |

### 导入方法
1. 将音效文件拖入 `Assets/Audio` 文件夹
2. 或右键 `Audio` → `Import New Asset...`

---

## 🎵 第二步：配置背景音乐

### 在Unity中操作
1. **Hierarchy** → 右键 → `Create Empty` → 命名为 `BGMPlayer`
2. 选中 `BGMPlayer` → `Add Component` → `Audio Source`
3. 配置参数：
   ```
   AudioClip: [拖入背景音乐文件]
   Play On Awake: ✓
   Loop: ✓
   Volume: 0.3 - 0.5
   Spatial Blend: 0
   ```

---

## 🎮 第三步：配置玩家音效

### 在Unity中操作
1. **Hierarchy** → 选择 `Player` 对象
2. **Inspector** → 找到 `Player Controller (Script)`
3. 在 **音效设置** 部分：
   ```
   Jump Clip: [拖入跳跃音效]
   Land Clip: [拖入落地音效]
   Sound Volume: 0.8
   ```

---

## 💰 第四步：配置金币音效

### 在Unity中操作
1. 找到金币预制体或场景中的金币对象
2. **Inspector** → `Collectible (Script)`
3. 配置：
   ```
   Pickup Sound: [拖入拾取音效]
   Sound Volume: 0.8
   ```

---

## ⚠️ 第五步：配置陷阱音效

### 在Unity中操作
1. **Hierarchy** → 选择陷阱对象
2. **Inspector** → `Hazard (Script)`
3. 配置：
   ```
   Hazard Sound: [拖入陷阱音效]
   Sound Volume: 0.8
   ```

---

## ✨ 第六步：创建粒子特效

### 落地尘土特效
1. **Hierarchy** → 右键 → `Effects` → `Particle System`
2. 命名为 `FX_LandDust`
3. 关键参数：
   ```
   Duration: 0.5
   Looping: ✗
   Start Lifetime: 0.5
   Start Speed: 2
   Start Size: 0.3
   Start Color: 灰褐色
   
   Emission → Bursts:
     Time: 0
     Count: 15
   
   Shape:
     Shape: Circle
     Radius: 0.5
   ```

### 金币拾取特效
1. 创建 `Particle System` → 命名为 `FX_CoinCollect`
2. 关键参数：
   ```
   Duration: 1.0
   Looping: ✗
   Start Color: 金黄色
   Start Size: 0.2
   Start Speed: 3
   
   Emission → Bursts:
     Count: 20
   
   Shape:
     Shape: Sphere
     Radius: 0.3
   ```

### 陷阱触发特效
1. 创建 `Particle System` → 命名为 `FX_HazardHit`
2. 关键参数：
   ```
   Start Color: 红色/橙色
   Start Size: 0.15
   Start Speed: 5
   
   Emission → Bursts:
     Count: 25
   
   Shape:
     Shape: Cone
     Angle: 30
   ```

## ⚠️ 常见问题

### 听不到声音？
- 检查主摄像机是否有 `Audio Listener` 组件
- 检查电脑音量
- 检查 Unity 编辑器静音按钮

### 特效不显示？
- 检查预制体是否正确拖入
- 检查 `Looping` 是否取消勾选
- 检查特效颜色是否设置了透明度

### 音效太大/太小？
- 调整 `Sound Volume` 参数（0-1）
- 调整 `Audio Source` 的 `Volume`
