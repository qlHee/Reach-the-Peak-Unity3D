## 📥 任务1：准备音效素材

### 1.1 下载音效资源

**推荐网站（免费音效）：**
- **Freesound.org** - https://freesound.org/
- **Mixkit** - https://mixkit.co/free-sound-effects/
- **Zapsplat** - https://www.zapsplat.com/
- **Unity Asset Store** - https://assetstore.unity.com/ (搜索 "Free Sound Effects")

**需要下载的音效类型：**
1. **背景音乐** (BGM) - 格式：.mp3 或 .wav，时长：1-3分钟，循环播放
2. **跳跃音效** - 短促的"嗖"声或"跳"声
3. **落地音效** - "咚"声或脚步声
4. **拾取金币音效** - 清脆的"叮"声或"铃"声
5. **陷阱触发音效** - "嘶"声、爆炸声或警报声
6. **环境音效**（可选）- 风声、水流声、虫鸣声

### 1.2 导入音效到Unity

**步骤：**
1. 在Unity编辑器中，找到 `Assets/Audio` 文件夹
2. 将下载的音效文件直接拖入该文件夹
3. 或者右键点击 `Audio` 文件夹 → `Import New Asset...` → 选择音频文件

**音频格式建议：**
- **短音效**（跳跃、拾取等）：`.wav` 格式，未压缩，质量最佳
- **背景音乐**：`.mp3` 格式，压缩格式，节省空间

**音频导入设置（在Inspector面板）：**
- **Load Type**: 
  - 短音效：`Decompress On Load`（立即解压）
  - 背景音乐：`Streaming`（流式播放）
- **Compression Format**: 
  - 短音效：`PCM`（无损）
  - 背景音乐：`Vorbis`（压缩）

## 🎵 任务2：添加背景音乐与环境音效

### 2.1 创建背景音乐播放器

**步骤：**
1. 在 **Hierarchy** 面板中，右键 → `Create Empty`
2. 重命名为 `BGMPlayer`
3. 选中 `BGMPlayer`，在 **Inspector** 面板点击 `Add Component`
4. 搜索并添加 `Audio Source` 组件

### 2.2 配置 AudioSource 组件

**在 Inspector 面板中设置：**
- **AudioClip**: 将你的背景音乐文件从 `Assets/Audio` 拖入此字段
- **Play On Awake**: ✓ 勾选（场景开始时自动播放）
- **Loop**: ✓ 勾选（循环播放）
- **Volume**: 0.3 ~ 0.5（根据需要调整，避免过大）
- **Spatial Blend**: 0（2D音效，不受距离影响）

### 2.3 添加环境音效（可选）

**步骤：**
1. 创建新的空对象，命名为 `EnvironmentSound`
2. 添加 `Audio Source` 组件
3. 导入环境音效（如风声、水流声）
4. 设置：
   - **Loop**: ✓ 勾选
   - **Volume**: 0.2 ~ 0.3（比BGM更低）
   - **Spatial Blend**: 0.5 ~ 1.0（可选3D音效）

## 🎮 任务3：为玩家添加音效

### 3.1 配置 PlayerController 音效

**步骤：**
1. 在 **Hierarchy** 中选择你的 **Player** 对象
2. 在 **Inspector** 面板找到 `Player Controller (Script)` 组件
3. 你会看到新增的 **音效设置** 和 **特效设置** 部分

**配置音效：**
- **Jump Clip**: 将跳跃音效文件拖入此字段
- **Land Clip**: 将落地音效文件拖入此字段
- **Sound Volume**: 0.8（调整音量）

**配置特效（稍后完成）：**
- **Land Effect Prefab**: 留空，等创建粒子特效后再配置

### 3.2 测试跳跃和落地音效

**步骤：**
1. 点击 Unity 编辑器顶部的 **Play** 按钮
2. 按 **空格键** 跳跃，应该听到跳跃音效
3. 落地时应该听到落地音效
4. 如果没有声音，检查：
   - 主摄像机是否有 `Audio Listener` 组件
   - 音效文件是否正确拖入
   - 电脑音量是否打开

## ✨ 任务6：创建粒子特效

### 6.1 创建落地尘土特效

**步骤：**
1. 在 **Hierarchy** 面板，右键 → `Effects` → `Particle System`
2. 重命名为 `FX_LandDust`（落地尘土）
3. 选中该对象，在 **Inspector** 面板配置：

**基础设置：**
- **Duration**: 0.5（特效持续时间）
- **Looping**: ✗ 取消勾选（不循环）
- **Start Lifetime**: 0.5
- **Start Speed**: 2
- **Start Size**: 0.3
- **Start Color**: 灰褐色（模拟尘土）

**Emission（发射）模块：**
- **Rate over Time**: 0
- **Bursts**: 点击 `+` 添加爆发
  - **Time**: 0
  - **Count**: 10-20（粒子数量）

**Shape（形状）模块：**
- **Shape**: Circle（圆形）
- **Radius**: 0.5
- **Emit from**: Edge（从边缘发射）

**Color over Lifetime（颜色渐变）：**
- 启用此模块
- 设置渐变：从不透明 → 透明（Alpha: 255 → 0）

**Size over Lifetime（大小渐变）：**
- 启用此模块
- 设置曲线：从小到大再到小

### 6.2 创建金币拾取特效

**步骤：**
1. 创建新的 `Particle System`，命名为 `FX_CoinCollect`
2. 配置：

**基础设置：**
- **Duration**: 1.0
- **Looping**: ✗ 取消勾选
- **Start Lifetime**: 0.8
- **Start Speed**: 3
- **Start Size**: 0.2
- **Start Color**: 金黄色

**Emission：**
- **Bursts**: Count = 15-25

**Shape：**
- **Shape**: Sphere（球形）
- **Radius**: 0.3

**Color over Lifetime：**
- 金黄色 → 透明

### 6.3 创建陷阱触发特效

**步骤：**
1. 创建新的 `Particle System`，命名为 `FX_HazardHit`
2. 配置：

**基础设置：**
- **Start Color**: 红色或橙色（火花效果）
- **Start Size**: 0.15
- **Start Speed**: 5

**Emission：**
- **Bursts**: Count = 20-30

**Shape：**
- **Shape**: Cone（锥形）
- **Angle**: 30

### 6.4 将特效制作成预制体

**重要步骤：**
1. 在 **Project** 面板中，打开 `Assets/Effects` 文件夹
2. 将 **Hierarchy** 中的特效对象（如 `FX_LandDust`）拖入 `Effects` 文件夹
3. 这样就创建了预制体（Prefab），可以重复使用
4. 对所有特效重复此操作
5. 创建完预制体后，可以从 **Hierarchy** 中删除原始对象

## 🧪 任务8：测试所有功能

### 8.1 测试清单

**点击 Play 按钮，依次测试：**

- [ ] **背景音乐**：场景开始时自动播放
- [ ] **跳跃音效**：按空格键跳跃时播放
- [ ] **落地音效**：角色落地时播放
- [ ] **落地特效**：落地时出现尘土粒子
- [ ] **拾取音效**：碰到金币时播放
- [ ] **拾取特效**：拾取金币时出现粒子效果
- [ ] **陷阱音效**：触碰陷阱时播放
- [ ] **陷阱特效**：触碰陷阱时出现粒子效果

### 8.2 常见问题排查

**问题1：听不到任何声音**
- 检查主摄像机是否有 `Audio Listener` 组件
- 检查电脑音量是否打开
- 检查 Unity 编辑器右上角的静音按钮是否关闭

**问题2：音效太大或太小**
- 调整各个脚本中的 `Sound Volume` 参数（0-1之间）
- 调整 `Audio Source` 组件的 `Volume` 参数

**问题3：特效不显示**
- 检查预制体是否正确拖入脚本字段
- 检查特效的 `Start Color` 是否设置了透明度
- 检查特效的位置是否在摄像机视野内

**问题4：特效不消失**
- 检查 `Looping` 是否取消勾选
- 检查脚本中的 `effectDestroyTime` 参数
